## ğŸ“ Key Concepts Summary

### 1. **Story = Component + Props**
```
Story = Your Component + Specific Props (args)
```

### 2. **Meta = Configuration for All Stories**
```
Meta = {
    Where to show (title),
    What to show (component),
    How to wrap it (decorators),
    How to document it (tags, argTypes)
}
```

### 3. **Decorators = Wrappers**
```
Decorators provide context:
- Redux Provider â†’ for useSelector/useDispatch
- Router â†’ for useNavigate/Link
- Theme Provider â†’ for styled components
- Custom styling â†’ for better visualization
```

### 4. **Args = Props**
```
args: {
    prop1: value1,  â† Passed to component as props
    prop2: value2,
}

Same as:
<YourComponent prop1={value1} prop2={value2} />
```

4. **What states should I show?**
